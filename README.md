# Relatório Fiscal Consolidado por Lojas

Projeto desenvolvido em Python para consolidar o faturamento e o cálculo de impostos de diferentes lojas, aplicando regras de tributação e classificando a carga tributária total.

## Regras de imposto

- até R$1000 → 10%
- acima de R$1000 → 15%
- acima de R$1500 → 20%

## Funcionalidades

- cálculo automático de imposto por venda
- faturamento total por loja
- imposto total por loja
- faturamento geral consolidado
- identificação da loja com maior carga tributária
- classificação final da carga tributária

## Tecnologias utilizadas

- Python
- Estruturas de repetição
- Funções
- Dicionários
- Condicionais
- Tuplas (retorno de funções)

## Objetivo

Praticar lógica de programação, estruturação de código e organização de dados simulando um cenário real de controle fiscal empresarial.
